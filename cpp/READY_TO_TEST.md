# 🎉 SMOKE TEST MVP ГОТОВИЙ!

## Що реалізовано

### ✅ Повний функціонал MVP
- **Меню гри** з кнопками "START GAME" та "EXIT"
- **3 будівлі:** Преторій (Рим), Головне шатро (Карфаген), Контуберній (римська казарма)
- **2 типи юнітів:** Легіонер (Рим) та Фінікієць (Карфаген)
- **Інтерактивність:** Вибір об'єктів, створення юнітів, пересування
- **HUD:** Відображення ресурсів та інформації про вибрані об'єкти

### 🎮 Як грати
1. **Меню:** Натисніть "START GAME" для початку
2. **Вибір:** Лівий клік для вибору будівель та юнітів
3. **Створення юнітів:** 
   - Клік на римську казарму (червоний прямокутник) → створює легіонера
   - Клік на карфагенське головне шатро (синій прямокутник) → створює фінікійця
4. **Пересування:** Правий клік для переміщення вибраного юніта
5. **Повернення:** ESC для повернення в меню

## 🚀 Запуск гри

### Крок 1: Встановити Raylib
1. Йдіть на https://raylib.com/
2. Натисніть "Download" → "Windows Installer"
3. Встановіть в `C:/raylib/` (включає MinGW компілятор)

### Крок 2: Компіляція
```bash
# У папці cpp/
.\compile.bat
```

### Крок 3: Запуск
```bash
# Якщо компіляція успішна
.\run.bat

# Або напряму
.\punic_wars.exe
```

## 🔧 Альтернативна компіляція

Якщо `compile.bat` не працює, спробуйте вручну:
```bash
g++ src/main.cpp -o punic_wars.exe -lraylib -lopengl32 -lgdi32 -lwinmm
```

## 🎯 Очікувані результати

Після запуску ви побачите:
1. **Чорне меню** з золотим заголовком та зеленими/червоними кнопками
2. **Зелену карту** з червоними (Рим) та синіми (Карфаген) будівлями
3. **HUD зверху** з ресурсами обох фракцій

## 🐛 Можливі проблеми

### "g++ not found"
- Переконайтеся, що Raylib встановлений з MinGW
- Перезапустіть командний рядок після встановлення

### "raylib not found"
- Перевірте, що Raylib встановлений в системний PATH
- Спробуйте перезавантажити комп'ютер

### Гра не запускається
- Перевірте, що файл `punic_wars.exe` створився після компіляції
- Запустіть з командного рядка для перегляду помилок
