# Оновлення курсорів та панелі ресурсів

## Що було додано:

### 1. Кастомні курсори
- **Cursor_idle.png** - звичайний курсор
- **Cursor_hover.png** - курсор при наведенні на ресурси (для збору)
- **Cursor_click.png** - курсор при наведенні на ворогів (для атаки)

Курсори автоматично змінюються залежно від контексту:
- Звичайний курсор - коли нічого не вибрано або вибрані юніти без спеціальних дій
- Курсор збору (hover) - коли вибрані раби і наведено на ресурси
- Курсор атаки (click) - коли вибрані солдати і наведено на ворожих юнітів

### 2. Панель ресурсів
- **ResourcePanel.png** - графічна панель для відображення ресурсів
- Панель відображається зверху по центру екрану
- Показує кількість їжі (ліворуч) та грошей (праворуч)
- Якщо панель не завантажиться, використовується fallback з текстом

## Технічні деталі:

### Завантаження текстур:
```cpp
cursorIdle = LoadTexture("assets/sprites/Cursor_idle.png");
cursorHover = LoadTexture("assets/sprites/Cursor_hover.png");
cursorClick = LoadTexture("assets/sprites/Cursor_click.png");
resourcePanel = LoadTexture("assets/sprites/ResourcePanel.png");
```

### Системний курсор прихований:
```cpp
HideCursor();
```

### Малювання курсора:
Функція `DrawCustomCursor()` викликається в кінці кожного кадру для всіх станів гри (меню, налаштування, вибір фракції, гра).

### Малювання панелі ресурсів:
Функція малює панель по центру зверху екрану з накладеними значеннями ресурсів.

## Як протестувати:

1. Запустіть гру: `./punic_wars.exe`
2. Перейдіть в меню і виберіть фракцію
3. В грі:
   - Перевірте, що курсор відображається правильно
   - Виберіть раба і наведіть на ресурс - курсор має змінитися
   - Виберіть солдата і наведіть на ворога - курсор має змінитися
   - Перевірте панель ресурсів зверху екрану

## Позиції тексту на панелі:

Якщо потрібно підлаштувати позиції цифр на панелі, змініть значення в `main.cpp`:
```cpp
// Їжа (ліворуч)
DrawText(TextFormat("%d", available_food), (int)panelX + 120, 35, 20, {255, 255, 200, 255});

// Гроші (праворуч)
DrawText(TextFormat("%d", available_money), (int)panelX + resourcePanel.width - 150, 35, 20, {255, 255, 200, 255});
```

Змініть `+ 120` та `- 150` для горизонтального зміщення, `35` для вертикального.
