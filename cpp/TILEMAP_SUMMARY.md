## 📊 Технічні характеристики

### Продуктивність
- ⚡ Генерація 50x50: < 100мс
- ⚡ Генерація 100x100: < 500мс
- ⚡ Рендеринг: 60 FPS з culling
- ⚡ Пам'ять: < 1 МБ для карти 100x100

### Математика
```
Grid → Screen:
  screenX = (col - row) * 64
  screenY = (col + row) * 32

Screen → Grid:
  col = (screenX / 64 + screenY / 32) / 2
  row = (screenY / 32 - screenX / 64) / 2
```

### Формат тайлсету
```
Розмір: 256x128 пікселів (сітка 2x2)
Тайл: 128x64 пікселів (ізометричний ромб)

[Трава] [Вода]
[Пісок] [Дорога]
```

## 🔧 Налаштування

### Параметри генерації
```cpp
GenerationParams params;
params.water_level = 0.3f;    // 0-30% = вода
params.sand_level = 0.5f;     // 30-50% = пісок
params.grass_level = 0.9f;    // 50-90% = трава
params.noise_scale = 0.1f;    // Масштаб деталей
params.octaves = 4;           // Деталізація (1-8)
```

### Розміри карти
- Мінімум: 10x10
- Рекомендовано: 50x50
- Максимум: 200x200 (автоматичне обмеження)
