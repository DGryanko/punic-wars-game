# Налаштування Raylib для Windows

## Крок 1: Встановлення Raylib

### Варіант 1: Офіційний інсталятор (рекомендовано)
1. Йдіть на https://raylib.com/
2. Натисніть "Download" → "Windows Installer"
3. Скачайте та встановіть в `C:/raylib/`
4. Інсталятор включає MinGW компілятор та Notepad++

### Варіант 2: Ручне встановлення
1. Скачайте MinGW-w64 з https://www.mingw-w64.org/
2. Скачайте Raylib з https://github.com/raysan5/raylib/releases
3. Розпакуйте та налаштуйте PATH

## Крок 2: Перевірка встановлення

Відкрийте командний рядок (cmd) та виконайте:
```bash
g++ --version
```

Повинно показати версію компілятора.

## Крок 3: Компіляція тестового проєкту

У папці cpp/ виконайте:
```bash
# Швидка компіляція
make quick

# Або повна збірка
make

# Запуск
make run
```

## Альтернативні команди

Якщо make не працює, використовуйте пряму команду:
```bash
g++ src/main.cpp -o punic_wars.exe -lraylib -lopengl32 -lgdi32 -lwinmm
```

## Можливі проблеми

### Помилка: "raylib not found"
- Переконайтеся, що Raylib встановлений в системний PATH
- Або додайте `-I/path/to/raylib/include -L/path/to/raylib/lib`

### Помилка: "g++ not found"
- Встановіть MinGW або додайте його в PATH
- Перезапустіть командний рядок після встановлення

### Помилка компонування
- Переконайтеся, що всі бібліотеки (-lraylib -lopengl32 -lgdi32 -lwinmm) вказані
- Порядок бібліотек може бути важливим
