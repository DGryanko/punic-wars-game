# Швидкий старт - Будівництво рабом

## Як протестувати нову функцію

### Крок 1: Запустіть гру
```bash
cd cpp
.\run.bat
```

### Крок 2: Почніть нову гру
1. Натисніть "START GAME"
2. Виберіть "ROME"

### Крок 3: Створіть раба
1. Клікніть на Преторій (червона будівля в центрі)
2. Натисніть кнопку "SLV" (Slave) в панелі знизу
3. Зачекайте 6 секунд поки раб з'явиться

### Крок 4: Виберіть раба
1. Клікніть на жовтого юніта (раба)
2. Знизу зліва з'явиться панель "Slave - Build"

### Крок 5: Побудуйте будівлю
Ви побачите дві кнопки:

**Квесторіум (QST)** - Склад ресурсів
- Вартість: 20 їжі, 30 золота
- Колір: темно-зелений
- Призначення: раби носять сюди ресурси

**Контуберній (CTB)** - Казарма
- Вартість: 30 їжі, 50 золота
- Колір: темно-синій
- Призначення: виробництво легіонерів

### Крок 6: Натисніть на кнопку
- Будівля з'явиться на відстані 2-3 тайли від раба
- Ресурси будуть витрачені автоматично

## Початкові ресурси

**Рим стартує з:**
- 200 їжі
- 100 золота

**Після створення раба (10/20):**
- 190 їжі
- 80 золота

**Можна побудувати:**
- ✅ Квесторіум (20/30) - залишиться 170 їжі, 50 золота
- ✅ Контуберній (30/50) - залишиться 160 їжі, 30 золота

## Поради

1. **Спочатку побудуйте квесторіум** - він дешевший і потрібен для збору ресурсів
2. **Потім контуберній** - для виробництва легіонерів
3. **Створіть більше рабів** - для швидшого збору ресурсів
4. **Призначте рабів на ресурси** - правий клік по ресурсу (жовті/сині точки на карті)

## Керування

- **ЛКМ** - вибір юніта/будівлі
- **ПКМ** - рух/атака/збір ресурсів
- **WASD / Стрілки** - рух камери
- **Коліщатко миші** - зум
- **ESC** - меню паузи

## Що далі?

Після будівництва:
1. Створіть більше рабів для збору ресурсів
2. Призначте їх на ресурсні точки (ПКМ по жовтих/синіх точках)
3. Побудуйте легіонерів в контуберній
4. Захищайте свою базу від ворогів!
