# Аудіо функції в Punic Wars: Castra

## Додані функції (Сесія 3)

### 1. Фонова музика
- **Меню**: `Punic wars_ Castra.mp3` - грає в головному меню та екрані вибору фракції
- **Бій (Рим)**: `Punic wars_ Castra Battlemusic Rome.mp3` - грає коли гравець обирає Рим
- **Бій (Карфаген)**: `Punic wars_ Castra Battlemusic Carthage.mp3` - грає коли гравець обирає Карфаген

### 2. Фоновий малюнок меню
- `Background.png` - відображається в головному меню з напівпрозорим оверлеєм

### 3. Меню налаштувань
Нова кнопка "SETTINGS" в головному меню відкриває екран налаштувань з трьома повзунками:

#### Повзунки гучності:
- **Music Volume** (Музика) - контролює гучність фонової музики (0-100%)
- **Ambient Volume** (Середовище) - зарезервовано для звуків середовища (0-100%)
- **Effects Volume** (Ефекти) - зарезервовано для звукових ефектів (0-100%)

### 4. Керування музикою
- Музика автоматично перемикається між меню та грою
- При виборі фракції запускається відповідна бойова музика
- При натисканні ESC в грі повертається меню музика
- Гучність застосовується в реальному часі

## Технічні деталі

### Використані Raylib функції:
- `InitAudioDevice()` - ініціалізація аудіо системи
- `LoadMusicStream()` - завантаження музичних файлів
- `PlayMusicStream()` - запуск музики
- `StopMusicStream()` - зупинка музики
- `UpdateMusicStream()` - оновлення музики в головному циклі
- `SetMusicVolume()` - встановлення гучності
- `UnloadMusicStream()` - вивантаження музики
- `LoadTexture()` - завантаження фону
- `DrawTexture()` - малювання фону

### Структура AudioSettings:
```cpp
struct AudioSettings {
    float musicVolume = 0.5f;      // Музика (0.0 - 1.0)
    float ambientVolume = 0.5f;    // Середовище (0.0 - 1.0)
    float effectsVolume = 0.5f;    // Ефекти (0.0 - 1.0)
};
```

## Наступні кроки

### Можливі покращення:
- [ ] Додати звукові ефекти для дій (атака, виробництво, збір ресурсів)
- [ ] Додати звуки середовища (вітер, птахи)
- [ ] Зберігати налаштування звуку в файл
- [ ] Додати можливість вимкнути музику повністю
- [ ] Додати fade in/fade out при перемиканні треків
- [ ] Використати Peaceful треки для мирного режиму

## Використання

1. **Запуск гри**: Меню музика грає автоматично
2. **Налаштування**: Натисніть "SETTINGS" в меню
3. **Регулювання гучності**: Перетягуйте повзунки мишкою
4. **Повернення**: Натисніть "BACK" для повернення в меню
5. **Гра**: Виберіть фракцію - запуститься відповідна бойова музика
6. **Вихід з гри**: ESC повертає в меню з меню музикою
