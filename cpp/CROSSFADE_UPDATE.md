# Оновлення: Плавне перемикання музики (Crossfade)

## Дата: 28 лютого 2026

## Зміни

### 1. Плавне перемикання між музичними треками (Crossfade)

Додано систему плавного переходу між музичними треками з тривалістю 2 секунди.

#### Нові змінні:
```cpp
Music nextMusic;                // Наступний трек для crossfade
bool isCrossfading = false;
float crossfadeTimer = 0.0f;
const float CROSSFADE_DURATION = 2.0f; // 2 секунди для плавного переходу
```

#### Нові функції:

**UpdateCrossfade()** - Оновлює стан crossfade:
- Плавно зменшує гучність поточного треку
- Плавно збільшує гучність нового треку
- Після завершення переходу зупиняє старий трек

**SwitchMusic()** (оновлена) - Тепер використовує crossfade:
- Запускає новий трек з нульовою гучністю
- Ініціює процес crossfade
- Обидва треки грають одночасно під час переходу

**UpdateGameMusic()** (оновлена):
- Оновлює обидва музичні потоки під час crossfade
- Викликає UpdateCrossfade() для обробки плавного переходу

### 2. Покращення логіки збору ресурсів

#### Проблеми, які були виправлені:

1. **Перевірка існування ресурсу**
   - Раби тепер перевіряють, чи ресурс ще існує перед збором
   - Якщо ресурс вичерпано, призначення скидається автоматично

2. **Покращена логіка руху**
   - Раби автоматично йдуть до будівлі для здачі, коли повні
   - Після здачі автоматично повертаються до ресурсу
   - Якщо раб не біля ресурсу, він автоматично йде до призначеної точки

3. **Стабільність циклу збору**
   - Виправлено проблему, коли раби "застрявали" після здачі ресурсів
   - Додано перевірку відстані для автоматичного руху до ресурсу
   - Покращено логіку перемикання між станами (збір/рух/здача)

#### Оновлена функція ProcessResourceHarvesting():

```cpp
// Основні покращення:
1. Перевірка існування ресурсу перед збором
2. Автоматичний рух до будівлі при повному інвентарі
3. Автоматичне повернення до ресурсу після здачі
4. Автоматичний рух до ресурсу, якщо раб далеко від нього
```

## Як це працює

### Crossfade:
1. При зміні музичного контексту (бій, мир, напруга) викликається SwitchMusic()
2. Новий трек запускається з гучністю 0%
3. Протягом 2 секунд:
   - Старий трек: 100% → 0%
   - Новий трек: 0% → 100%
4. Після завершення старий трек зупиняється

### Збір ресурсів:
1. Гравець призначає раба до ресурсу (ПКМ по ресурсу)
2. Раб йде до ресурсу і починає збір
3. Коли інвентар повний, раб автоматично йде до найближчої будівлі
4. Після здачі ресурсів раб автоматично повертається до ресурсу
5. Цикл повторюється до вичерпання ресурсу

## Використання

### Для тестування crossfade:
1. Запустіть гру
2. Виберіть фракцію (почнеться амбієнт музика)
3. Створіть юнітів і почніть бій (музика плавно перейде на бойову)
4. Відійдіть від ворогів (музика плавно повернеться до амбієнту)

### Для тестування збору ресурсів:
1. Виберіть Рим як фракцію
2. Створіть рабів з HQ (кнопка SLV)
3. Виділіть рабів (ЛКМ або область)
4. ПКМ по ресурсу (зелений або жовтий квадрат)
5. Спостерігайте за автоматичним циклом збору

## Технічні деталі

### Продуктивність:
- Crossfade додає мінімальне навантаження (2 музичні потоки одночасно)
- Логіка збору ресурсів оптимізована для великої кількості рабів
- Немає помітного впливу на FPS

### Сумісність:
- Працює з усіма музичними треками
- Сумісно з існуючою системою налаштувань звуку
- Не впливає на інші системи гри
