# Система будівництва рабом

## Опис функціональності

Додано нову систему будівництва, яка дозволяє рабам будувати квесторіум (склад ресурсів) та контуберній (казарму) для Риму.

## Як використовувати

1. **Виберіть раба** - клікніть лівою кнопкою миші на раба (жовтий юніт з позначкою "SLV")

2. **Відкриється панель будівництва** - знизу зліва з'явиться панель "Slave - Build" з двома кнопками

3. **Перевірте ресурси**:
   - **Квесторіум (QST)**: 20 їжі, 30 золота
   - **Контуберній (CTB)**: 30 їжі, 50 золота

4. **Побудуйте будівлю** - натисніть на відповідну кнопку:
   - Будівля з'явиться на відстані 2-3 тайли від раба
   - Система автоматично знайде вільне місце
   - Ресурси будуть витрачені одразу

## Вартість будівель

### Квесторіум (Склад ресурсів)
- 20 їжі (Food)
- 30 золота (Money)
- Колір кнопки: темно-зелений

### Контуберній (Казарма)
- 30 їжі (Food)
- 50 золота (Money)
- Колір кнопки: темно-синій

## Технічні деталі

### Нові файли
- `cpp/src/slave_build_panel.h` - UI панель для будівництва рабом

### Зміни в існуючих файлах
- `cpp/src/main.cpp`:
  - Додано include для `slave_build_panel.h`
  - Додано глобальну змінну `slaveBuildPanel`
  - Ініціалізація панелі в main()
  - Обробка кліків по панелі в HandleClicks()
  - Відображення панелі в DrawGame()
  - Оновлення selectedUnitIndex при виборі раба

### Логіка розміщення
- Обидві будівлі займають 2x2 тайли
- Шукається вільне місце на відстані 2-3 тайли від раба
- Перевіряється що місце не зайняте іншими будівлями
- Перевіряються межі карти (80x80 тайлів)

### Вартість будівництва
```cpp
BuildingCost getBuildingCost(BuildingType type) {
    if (type == QUESTORIUM_ROME) {
        return {20, 30};  // 20 їжі, 30 золота
    } else if (type == BARRACKS_ROME) {
        return {30, 50};  // 30 їжі, 50 золота (Контуберній)
    }
    return {0, 0};
}
```

## UI елементи

### Панель будівництва
- Розташування: знизу зліва (10, 950, 300x100)
- Заголовок: "Slave - Build"
- Дві кнопки: 50x50 пікселів кожна
- Відображення вартості під кожною кнопкою
- Колір кнопок:
  - Зелений/Синій - якщо достатньо ресурсів
  - Сірий (DARKGRAY) - якщо недостатньо ресурсів

### Іконки
- **QST** - Квесторіум (тимчасово текст)
- **CTB** - Контуберній (тимчасово текст)
- В майбутньому можна додати спрайти будівель

## Майбутні покращення

1. **Додати анімацію будівництва** - раб йде до місця будівництва
2. **Додати час будівництва** - будівлі будуються не миттєво
3. **Додати інші будівлі** - можливість будувати інші типи будівель
4. **Додати іконки** - замість тексту використовувати спрайти
5. **Додати звуки** - звук будівництва та завершення
6. **Додати для Карфагену** - аналогічна система для карфагенських рабів
7. **Візуальний фідбек** - показувати де буде побудована будівля

## Тестування

Для тестування функціональності:

1. Запустіть гру: `.\run.bat` (в папці cpp)
2. Виберіть фракцію Рим
3. Побудуйте раба в HQ (кнопка "SLV" в панелі HQ)
4. Зачекайте поки раб з'явиться (6 секунд)
5. Виберіть раба після його появи
6. Натисніть кнопку "QST" або "CTB" в панелі будівництва
7. Перевірте що будівля з'явилась поруч з рабом

## Баланс ресурсів

Початкові ресурси Риму: 200 їжі, 100 золота

- **Раб**: 10 їжі, 20 золота (залишається: 190 їжі, 80 золота)
- **Квесторіум**: 20 їжі, 30 золота (можна побудувати після раба!)
- **Контуберній**: 30 їжі, 50 золота (можна побудувати після раба!)

Тепер вартість збалансована і можна побудувати будівлі одразу після створення раба!

## Відомі обмеження

- Працює тільки для Риму (QUESTORIUM_ROME, BARRACKS_ROME)
- Немає анімації будівництва
- Немає перевірки чи раб може дійти до місця будівництва
- Будівлі з'являються миттєво
- Немає візуального попереднього перегляду місця будівництва
