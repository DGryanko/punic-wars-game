#### Порядок малювання (Z-order)
1. Спочатку центральна частина (base)
2. Потім ліва частина (перекриває центр)
3. Потім права частина (перекриває центр)
4. Зверху текст з обведенням

#### Обведення тексту
```cpp
// Чорне обведення (8 напрямків)
for (int ox = -2; ox <= 2; ox++) {
    for (int oy = -2; oy <= 2; oy++) {
        if (ox != 0 || oy != 0) {
            DrawText(text, x + ox, y + oy, size, BLACK);
        }
    }
}
// Білий текст зверху
DrawText(text, x, y, size, WHITE);
```

```cpp
float centerX = 1434.0f / 2.0f;
```
