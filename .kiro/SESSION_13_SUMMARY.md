## 🔧 Технічні деталі

### Поточні параметри
- **Тайл**: 128x64 пікселів (CoordinateConverter)
- **NavigationGrid**: 16x16 пікселів на клітинку (не синхронізовано!)
- **Текстура будівлі**: 384x224 пікселів
- **texture_offset**: {-192, -160}
- **Розмір мапи**: 80x80 тайлів

### Формули конвертації (CoordinateConverter)
```cpp
// Grid → Screen
screenX = (col - row) * 64
screenY = (col + row) * 32

// Screen → Grid
col = (screenX / 64 + screenY / 32) / 2
row = (screenY / 32 - screenX / 64) / 2
```

### Проблема з NavigationGrid
```cpp
// pathfinding.h:44-50
gridX = worldX / 16;  // cellSize = 16
gridY = worldY / 16;
```
Це НЕ відповідає ізометричній геометрії тайлів 128x64!
