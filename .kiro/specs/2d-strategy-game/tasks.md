# План реалізації: "Пунічні війни: Табори"

## Огляд

Поетапна реалізація 2D RTS гри на C++ з використанням Raylib. План слідує пріоритетам від Грока: від базового вікна до повноцінної гри з кампанією.

## Примітки

- Завдання позначені `*` є опціональними та можуть бути пропущені для швидшого MVP
- Кожне завдання посилається на конкретні вимоги для відстеження
- Checkpoint'и забезпечують поступову валідацію
- Property тести валідують універсальні властивості коректності
- Unit тести валідують конкретні приклади та граничні випадки
