## РЕАЛІЗОВАНІ ФУНКЦІЇ

### 1. МЕНЮ ТА ВИБІР ФРАКЦІЇ
- Головне меню з кнопками START GAME / EXIT
- Екран вибору фракції (ROME / CARTHAGE)
- Гравець керує тільки своєю фракцією

### 2. БУДІВЛІ (5 типів)
- **HQ_ROME** (Praetorium) - головна будівля Риму, виробляє рабів
- **HQ_CARTHAGE** (Main Tent) - головна будівля Карфагену
- **BARRACKS_ROME** (Contubernium) - казарма Риму, виробляє легіонерів (макс 8)
- **BARRACKS_CARTHAGE** (Mercenary Camp) - казарма Карфагену, виробляє фінікійців
- **QUESTORIUM_ROME** - склад для здачі ресурсів

### 3. ЮНІТИ (3 типи)
**Легіонер (Legionary)**
- HP: 100
- Урон: 25
- Дальність атаки: 30
- Перезарядка: 1.5с
- Швидкість: 0.85f
- Вартість: 30 їжі, 50 грошей
- Час виробництва: 10с

**Фінікієць (Phoenician)**
- HP: 90
- Урон: 30
- Дальність атаки: 25
- Перезарядка: 1.2с
- Швидкість: 0.85f
- Вартість: 25 їжі, 60 грошей
- Час виробництва: 9с

**Раб (Slave)**
- HP: 50
- Урон: 5
- Дальність атаки: 15
- Перезарядка: 2.0с
- Швидкість: 0.6f
- Вартість: 10 їжі, 20 грошей
- Час виробництва: 6с
- **Може збирати ресурси**

### 4. СИСТЕМА РЕСУРСІВ
- **FOOD_SOURCE** - точки їжі (зелені квадрати з "F")
- **GOLD_SOURCE** - точки золота (жовті квадрати з "G")
- Обмежена кількість ресурсів у кожній точці
- Раби автоматично носять ресурси до HQ/Квесторію

### 5. ВИРОБНИЦТВО ЮНІТІВ
- **Черга виробництва** - можна замовляти кілька юнітів
- **ЛКМ на іконці** - додати юніт до черги
- **ПКМ на іконці** - скасувати останній у черзі (повертає ресурси)
- **Число в куточку** - показує кількість у черзі
- **Прогрес-бар** - показує прогрес виробництва

### 6. БОЙОВА СИСТЕМА
- Юніти атакують ворогів у радіусі дії
- **Автоматична атака** - при наближенні ворога
- **ПКМ по ворогу** - наказ атакувати конкретну ціль
- Мертві юніти (HP = 0) видаляються з гри
- Візуальні індикатори здоров'я

### 7. КОЛІЗІЇ
- **Юніти + будівлі** - юніти не проходять крізь будівлі
- **Юніти + юніти** - юніти не проходять один через одного
- Юніти зупиняються при колізії

### 8. КЕРУВАННЯ
**Виділення юнітів:**
- ЛКМ - вибрати один юніт
- Перетягування ЛКМ - виділити область
- Подвійний клік - виділити всі юніти цього типу

**Команди:**
- ПКМ на землі - рух
- ПКМ на ресурсі - збирати (для рабів)
- ПКМ на ворогу - атакувати (для воїнів)

**Курсори:**
- Зелений "H" - збір ресурсів
- Червоний "X" - атака

### 9. UI/UX
- **HUD** - показує ресурси гравця (їжа, гроші)
- **Панель замовлення** - з'являється при виборі будівлі
- **Інформація про вибране** - показує характеристики юніта/будівлі
- **Статистика** - кількість юнітів

### 10. AI
- Ворожі юніти керуються AI
- Ворожі будівлі автоматично виробляють юніти
- Прості патрулі та атаки

### 11. АУДІО СИСТЕМА (НОВЕ!)
- **Фонова музика меню** - грає в головному меню та екрані вибору фракції
- **Бойова музика** - унікальна для кожної фракції (Рим/Карфаген)
- **Автоматичне перемикання** - музика змінюється при переході між меню та грою
- **Фоновий малюнок** - красивий фон в головному меню

### 12. МЕНЮ НАЛАШТУВАНЬ (НОВЕ!)
- **Music Volume** - повзунок гучності музики (0-100%)
- **Ambient Volume** - повзунок гучності середовища (0-100%)
- **Effects Volume** - повзунок гучності ефектів (0-100%)
- **Реальний час** - зміни застосовуються миттєво
- **Кнопка SETTINGS** - доступна з головного меню

### Система черги виробництва (building.h)

### Бойова система (unit.h)
```cpp
int attack_damage;         // Урон атаки
float attack_range;        // Дальність атаки
float attack_cooldown;     // Час між атаками
bool attackTarget(Unit& target); // Атакувати ціль
void takeDamage(int damage);     // Отримати урон
bool isDead() const;             // Перевірка смерті
```

### Колізії (main.cpp)
```cpp
// Перевірка колізій з будівлями
for (const auto& building : buildings) {
    if (CheckCollisionRecs(unitRect, building.getRect())) {
        hasCollision = true;
    }
}

// Перевірка колізій з іншими юнітами
for (int j = 0; j < units.size(); j++) {
    if (i != j) {
        float distance = sqrt(pow(units[i].x - units[j].x, 2) + 
                            pow(units[i].y - units[j].y, 2));
        if (distance < 16) hasCollision = true;
    }
}
```
